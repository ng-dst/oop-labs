\doxysubsection{Detailed Description}
\mbox{\hyperlink{classDungeonGame_1_1Level}{Level}} -\/ contains map and list of mobs on the level

Each level has its number and title. Numbers start with 0 and increment with each new level. \mbox{\hyperlink{classDungeonGame_1_1Level}{Level}} is loaded from two files\+:
\begin{DoxyItemize}
\item level\+X.\+ini -\/ level config and list of mobs
\item level\+\_\+map\+X.\+txt -\/ level map, denoted by the following symbols
\end{DoxyItemize}
